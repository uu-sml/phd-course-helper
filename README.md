# phdcourses

Course helper for SysCon PhD students.

To run locally, clone the repo and run 

`python3 -m http.server 8000`

to start hosting. Then visit

`http://localhost:8000`.

If anything is missing, wrong or inclomplete, please do no not hesitate to contact [the current maintainer](mailto:david.vavinggren@it.uu.se)! You can also make a pull request with your changes if you want to edit yourself.
